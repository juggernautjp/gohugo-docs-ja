1 つの使用例は、以下のような、[`delimit` 関数][`delimit` function] と組み合わせた要素の連結です。

{{< code file="slice.html" >}}
{{ $sliceOfStrings := slice "foo" "bar" "buzz" }}
<!-- スライス [ "foo", "bar", "buzz"] を返します -->
{{ delimit ($sliceOfStrings) ", " }}
<!-- 文字列 "foo, bar, buzz" を返します -->
{{< /code >}}
