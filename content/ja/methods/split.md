例:

```go-html-template
{{ split "tag1,tag2,tag3" "," }} → ["tag1", "tag2", "tag3"]
{{ split "abc" "" }} → ["a", "b", "c"]
```


{{< note >}}
`split` は基本的に、[delimit]({{< ref "functions/delimit" >}}) の逆を行います。 `split` が文字列からスライスを作成するのに対して、 `delimit` はスライスから文字列を作成します。
{{< /note >}}
